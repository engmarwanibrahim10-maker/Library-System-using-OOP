# 📚 Library Management System (C++)

A clean and efficient command-line tool to manage library operations. This project focuses on utilizing the C++ Standard Template Library (STL) to handle data storage and manipulation.

## ✨ Key Features
- **Real-time Inventory**: View all books currently available for borrowing.
- **Borrow & Return System**: Automatically moves books between available and borrowed lists.
- **Admin Capabilities**: Add new books to the library system during runtime.
- **Dynamic Storage**: Powered by `std::vector`, ensuring no fixed limit on the number of books.

## 🛠️ Technical Stack
- **Language**: C++
- **Containers**: `std::vector` (Standard Template Library)
- **Methodology**: Procedural logic within an Object-Oriented structure.
